# 🚀 MOVE Trip Planner

**Collaborative trip planning with real-time voting, budget tracking, and itinerary export**

## ✨ Features

- 🤝 **Real-time collaboration** - Plan with friends instantly
- 💰 **Budget tracking** - Never overspend
- 🗳️ **Smart voting** - Democratic decision making
- 📄 **PDF export** - Printable itineraries
- 🔍 **Smart search** - Find flights, hotels, activities

## 🎯 Quick Start

1. Create a trip with destination and dates
2. Invite friends (share the link)
3. Add flights, hotels, and activities
4. Vote on items together
5. Export your finalized itinerary

## 🙋‍♂️ How to Use

1. **Create Trip** - Enter destination, dates, group size, budget
2. **Add Items** - Search or add custom items to each day
3. **Vote** - Approve or veto items as a group
4. **Track Budget** - Watch your spending in real-time
5. **Export** - Download PDF itinerary when done
